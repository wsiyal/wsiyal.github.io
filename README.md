# Personal Website

A modern personal website built with React, Vite, and TailwindCSS.

## Features

- 🎨 Modern UI with TailwindCSS
- 📱 Fully responsive design
- ⚡ Fast performance with Vite
- 🌙 Dark mode support
- 📊 Research-focused content
- 🔧 Skills showcase
- 📱 Project portfolio

## Tech Stack

- **Framework:** React 18
- **Build Tool:** Vite
- **Styling:** TailwindCSS
- **Icons:** Lucide React

## Deployment

This website is deployed on GitHub Pages at:
`https://YOUR_USERNAME.github.io/personal-website/`

## Local Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Project Structure

```
src/
├── App.jsx          # Main application component
├── main.jsx         # Application entry point
├── index.css        # Global styles
└── lib/
    └── utils.js     # Utility functions
```
